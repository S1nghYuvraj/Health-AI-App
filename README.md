# 🏥 Health-AI-App

An AI-powered healthcare web application built using Flask, Machine Learning, OCR, PDF Processing, and Google Gemini AI. The platform provides disease prediction, health report analysis, and intelligent medical assistance through an easy-to-use web interface.

## 🚀 Features

### 🩺 Disease Prediction
- Predicts potential diseases using Machine Learning models.
- Supports multiple health-related prediction modules.
- Provides quick and accurate results.

### 📄 Medical Report Analysis
- Upload medical reports in PDF format.
- Extracts important health information automatically.
- Generates easy-to-understand summaries.

### 🔍 OCR-Based Text Extraction
- Extracts text from images and scanned medical documents.
- Supports health records and prescriptions.

### 🤖 AI Medical Assistant
- Powered by Google Gemini AI.
- Answers health-related queries.
- Provides personalized healthcare guidance.

### 📊 Health Data Processing
- Analyzes patient health information.
- Generates useful insights and recommendations.

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/S1nghYuvraj/Health-AI-App.git

cd Health-AI-App
```

### 2. Create Virtual Environment

```bash
python -m venv venv
```

Activate Environment

#### Windows

```bash
venv\Scripts\activate
```

#### macOS/Linux

```bash
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure Environment Variables

Create a `.env` file and add:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### 5. Run the Application

```bash
python app.py
```

Application will start at:

```text
http://127.0.0.1:5000
```
